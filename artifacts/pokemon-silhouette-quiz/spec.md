## Overview
"포켓몬 마스터 퀴즈" — 1세대 포켓몬 151종의 실루엣을 보고 4지선다로 이름을 맞추는 퀴즈 게임. 5문제 단위 라운드, 3단계 힌트, 연속 정답 보너스로 구성된다. 모바일 고정 크기(375x667px) 포켓몬 배틀 UI 스타일.

## 시나리오

### 1. 메인 화면
[상황] 사용자가 앱에 접속하면
[동작] 시스템이 포켓볼 일러스트, 타이틀, 게임 규칙을 표시한다

성공 기준:
- [ ] 포켓볼 SVG 일러스트가 중앙에 표시
- [ ] "포켓몬 마스터 퀴즈" 타이틀 + "1세대 포켓몬 151종" 부제 표시
- [ ] "포켓몬 마스터를 찾아라!" 텍스트박스 + "5문제 / 4지선다 / 힌트 3회" 규칙 안내
- [ ] '시작하기' 버튼 표시

### 2. 퀴즈 시작
[상황] 사용자가 메인 화면에서 '시작하기' 버튼을 누르면
[동작] 시스템이 1세대 포켓몬 중 랜덤 5종을 선택하고, 첫 번째 문제의 실루엣과 4개 보기를 표시한다

성공 기준:
- [ ] '시작하기' 버튼 클릭 → 실루엣 이미지 1개 + 보기 버튼 4개 표시
- [ ] 보기 4개 중 정답 1개 포함, 나머지 3개는 오답 (1세대 포켓몬 중 랜덤)
- [ ] 보기에 한국어/영어 두 줄로 가운데 정렬 표시 (예: "피카츄" + "(Pikachu)")
- [ ] 문제 번호 "1 / 5" + 진행률 도트 상단 고정 표시
- [ ] 점수 "0점" 우측 상단 고정 표시
- [ ] 포켓몬 이미지 위치가 문제 풀이 전후로 고정 (이동하지 않음)
- [ ] 보기 버튼 크기 60px 고정, 위치 불변

### 3. 정답 선택
[상황] 실루엣 문제가 표시된 상태에서 사용자가 정답 보기를 클릭하면
[동작] 시스템이 실루엣을 원본 이미지로 전환하고, "정답!" 메시지와 점수를 표시한 뒤 다음 문제로 넘어간다

성공 기준:
- [ ] 정답 클릭 → 실루엣이 컬러 이미지로 전환
- [ ] 정답 보기: 초록색 테두리 + 배경 + 체크 아이콘 + 살짝 확대
- [ ] 나머지 보기: opacity 30%로 흐리게
- [ ] "정답!" 텍스트 초록색 배경으로 표시
- [ ] 기본 점수 100점 획득, 점수 표시 갱신
- [ ] 연속 정답 시 콤보 표시 (예: "2 콤보!")
- [ ] 이미지 아래 고정 슬롯에 포켓몬 이름 표시
- [ ] 결과 확인 후 다음 문제로 자동 전환, 문제 번호 갱신 (예: "1 / 5" → "2 / 5")

### 4. 오답 선택
[상황] 실루엣 문제가 표시된 상태에서 사용자가 오답 보기를 클릭하면
[동작] 시스템이 실루엣을 컬러 이미지로 전환하고, 콤보를 초기화하고, 다음 문제로 넘어간다

성공 기준:
- [ ] 오답 클릭 → 선택한 보기에 빨간색 테두리 + 배경 + X 아이콘 + 취소선
- [ ] 정답 보기는 강조하지 않음 (다른 보기와 동일하게 흐리게 처리)
- [ ] 실루엣이 컬러 이미지로 전환 (정답 포켓몬 공개)
- [ ] "오답!" 텍스트 빨간색 배경으로 표시 + 정답 이름 안내
- [ ] 점수 변동 없음 (0점)
- [ ] 콤보 텍스트가 화면에서 사라진다 (콤보 초기화)
- [ ] 결과 확인 후 다음 문제로 자동 전환, 문제 번호 갱신

### 5. 힌트 사용
[상황] 실루엣 문제가 표시된 상태에서 사용자가 '힌트' 버튼을 누르면
[동작] 시스템이 단계별 힌트를 공개하고 획득 가능 점수를 감소시킨다

성공 기준:
- [ ] 힌트 1회 클릭 → 타입 정보 표시 (예: "전기 타입") + 점수 감소 표시 (~~100~~ → 70점 -30)
- [ ] 힌트 2회 클릭 → 타입 + 도감번호 표시 (예: "전기 타입  도감번호: #025") + 점수 감소 표시 (~~100~~ → 40점 -60)
- [ ] 힌트 3회 클릭 → 실루엣 밝기 50%로 변경 (반쯤 공개) + 점수 감소 표시 (~~100~~ → 20점 -80)
- [ ] 3회 사용 후 힌트 버튼 비활성화
- [ ] 힌트 표시 시 포켓몬 이미지 위치가 이동하지 않음

### 6. 결과 요약
[상황] 5문제를 모두 풀면
[동작] 시스템이 결과 요약 화면을 표시한다

성공 기준:
- [ ] 총점 표시 — 작은 접두사 "총점" + 큰 숫자 (예: "400점")
- [ ] 정답률 표시 — 작은 접두사 "정답률" + 큰 숫자 (예: "4 / 5") + 진행률 도트
- [ ] 최대 콤보 표시 — 작은 접두사 "콤보" + 큰 숫자 (예: "4")
- [ ] 3개 통계가 한 줄에 균등 분배 표시
- [ ] 틀린 포켓몬 목록 표시 (실루엣 → 정답 이미지 + 이름)
- [ ] '다시 하기' 버튼 — 클릭 시 홈 화면을 거치지 않고 바로 새 퀴즈 시작 (문제 번호 "1 / 5", 점수 "0점")
- [ ] '홈화면' 버튼 — 클릭 시 메인 화면으로 복귀

## 범위

### 포함
- 메인 화면 (포켓볼 일러스트 + 시작 버튼)
- 퀴즈 화면 (실루엣 + 4지선다 + 힌트 + 점수) — 모바일 고정 크기
- 결과 화면 (요약 + 다시 하기 + 홈화면)
- PokeAPI 연동 (1세대 포켓몬 데이터 + 스프라이트)
- 한국어/영어 두 줄 포켓몬 이름 표시
- 포켓볼 레드 테마 (primary: 빨간 포켓볼, secondary: 따뜻한 크림)
- 포켓몬 배틀 UI 스타일 (GameFrame, BattleField, MenuPanel, PokeballDivider)

### 제외
- 세대 선택 기능 — 1세대 고정으로 스코프를 줄이기 위해
- 리더보드/랭킹 — 백엔드 필요, 5시간 범위 초과
- 사용자 인증 — 과제 제출용으로 불필요
- 난이도 선택 — 단일 난이도로 완성도 집중
- 사운드 효과 — 구현 우선순위 낮음

## 전제 조건
- PokeAPI (https://pokeapi.co/)가 정상 동작하며, 1세대 포켓몬의 한국어 이름을 species 엔드포인트에서 제공한다
- 포켓몬 스프라이트 이미지가 official-artwork 형식으로 제공된다
- CSS `filter: brightness(0)`로 실루엣 효과를 구현할 수 있다
