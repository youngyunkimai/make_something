Notes

배틀 화면 스타일 퀴즈 진행 중. 힌트 1단계(타입)가 공개된 상태.

  • QUIZ-001: 배틀필드에 실루엣, 메뉴 패널에 보기 4개 (2x2 그리드)
  • QUIZ-002: 보기 중 정답 1개가 반드시 포함
  • QUIZ-007: 힌트 1회 → 배틀필드 하단 텍스트박스에 "전기 타입" + 획득 가능 70점
  • QUIZ-008: 힌트 2회 → "도감번호: #025" 추가 표시
  • QUIZ-009: 힌트 3회 → 실루엣 밝기 50% (silhouette-half 클래스), 힌트 버튼 비활성화
  • 선택된 보기 옆에 화살표(chevron-right) 표시 — RPG 메뉴 커서
  • HP바 스타일 점수 표시 (상단 우측 status-box)
  • 진행률은 도트(filled/empty)로 표시 — 포켓몬 배지 스타일
